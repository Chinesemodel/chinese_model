### 项目简介

**手写汉字识别**一个可以用鼠标在网站上书写汉字的平台,现如今各种输入法层出不穷，人们提笔忘字的几率也大大增加，这就产生了我们的手写汉字平台，可以让没事时间在纸上写字的人可以在电脑上写字。
